## قالب ADR

```markdown
# ADR-XXX: [العنوان]

## الحالة (Status)
[مقترح | مقبول | مرفوض | ملغى]

## السياق (Context)
[وصف المشكلة أو القرار المطلوب]

## الخيارات المدروسة (Options)
1. الخيار أ: [وصف] — [إيجابيات/سلبيات]
2. الخيار ب: [وصف] — [إيجابيات/سلبيات]
3. الخيار ج: [وصف] — [إيجابيات/سلبيات]

## القرار (Decision)
[الخيار المختار مع التبرير]

## العواقب (Consequences)
- إيجابيات: [...]
- سلبيات: [...]
- مخاطر: [...]

## المراجع (References)
- [روابط أو مستندات ذات صلة]
```
